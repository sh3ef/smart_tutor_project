# الإسلام

هُوَ شَهَادَةُ أَن لَا إِلَهَ إِلَّا اللَّهُ وَأَنَّ مُحَمَّدًا ﷺ رَسُولُ اللهِ،

# الإيمان

أَنْ نُؤْمِنَ بِاللَّهِ وَمَلائِكَتِهِ وَكُتُبِهِ، وَرُسُلِهِ وَالْيَوْمِ الآخِرِ،
وَالقَدَرِ خَيْرِهِ وَشَرِّهِ.

# الإحسان

أَنْ نَعْبُدَ اللَّهَ كَأَنَّنا نَراهُ، فَإِنْ لَمْ نَكُنْ نَرَاهُ فَإِنَّهُ يَرانا.

# نشاط

أَسْتَمِعُ إِلى زَمِيلِي فِي الْمَجْمُوعَةِ وَهُوَ يُسَمِّعُ مِنْ حِفْظِهِ تَعْرِيفَ الْإِسْلَامِ ، ثُمَّ
نَسْتَمِعُ إِلَى زَمِيلِنا الآخَرِ وَهُوَ يُعَرِّفُ الْإِيمَانَ ، ثُمَّ يَسْتَمِعُونَ إِلَيَّ وَأَنَا أُعَرِّفُ
الْإِحْسَانَ ، وَنُصَحْحُ أَخْطَاءَنَا حَالَ حُدُوثِهَا بِكُلِّ سُرُور.

# أَقْرَأُ، وَأُجِيبُ:

عَنِ ابْنِ عُمَرَ رَضِيَ اللَّهُ عَنْهُمَا قَالَ: قَالَ رَسُولُ اللَّهِ ﷺ :

# الشَّرطُ الثَّامِنُ: اسْتِقْبَالُ القِبْلَةِ.

أَضَعُ عَلامَةَ (✓) أَمَامَ كُلِّ عِبَارَةٍ لَهَا ارْتِبَاطٌ بِصُورَةِ الكَعْبَةِ:
*   تَقَعُ فِي مَكَّةَ المُكَرَّمَةِ. ( )
*   قِبْلَةُ المُسْلِمِينَ. ( )
*   تَقَعُ فِي المَدِينَةِ المُنَوَّرَةِ. ( )

# لا تصحُ الصَّلاةُ دُونَ اسْتِقْبَالِ القِبْلَةِ

# الشرط التاسع: النِّيَّةُ.

قال النَّبِيُّ ﷺ : «إِنَّمَا الْأَعْمَالُ بِالنِّيَّاتِ» (۱).
*   أَسْتَنْتِجُ مِن هَذَا الحَدِيثِ شَرْطًا مِن شُرُوطِ الصَّلاةِ: ..........

لا تَنْطِقُ بالنّيةِ، النِّيَّةُ مَحَلُّهَا القَلْبُ.
